# 📊 Vrinda Store Annual Sales Analysis (2022)

## 📌 Project Overview

This project presents an interactive sales dashboard built using Microsoft Excel to analyze the annual performance of **Vrinda Store** in 2022. The dashboard provides valuable insights into sales trends, customer demographics, order channels, and order status, enabling data-driven business decisions.

The analysis focuses on identifying top-performing states, understanding customer behavior, and evaluating sales across different platforms and product categories.

---

## 🎯 Project Objectives

- Analyze monthly sales and order trends.
- Compare total orders with total revenue.
- Identify the most profitable sales channels.
- Understand customer demographics based on age and gender.
- Determine the top-performing states by sales.
- Analyze order fulfillment status.
- Explore product category performance.

---

## 📂 Dataset Information

The dataset contains sales transactions from Vrinda Store for the year 2022.

### Dataset Features

| Column Name | Description |
|------------|-------------|
| Order ID | Unique identifier for each order |
| Order Date | Date of purchase |
| Amount | Total sales amount |
| Gender | Customer gender |
| Age Group | Customer age category |
| State | Customer state |
| Channel | Sales platform |
| Category | Product category |
| Order Status | Delivery status |

## 📊 Dashboard Features

### 📈 Orders vs Sales Analysis

- Tracks monthly sales and order volume.
- Compares revenue generated against the number of orders.
- Identifies seasonal sales patterns.

### 🛒 Order Channel Analysis

The dashboard analyzes sales across multiple channels:

- Amazon
- Flipkart
- Myntra
- Ajio
- Meesho
- Nalli
- Others

### 🏆 Top 5 States by Sales

The top-performing states based on revenue are:

1. Maharashtra
2. Karnataka
3. Uttar Pradesh
4. Telangana
5. Tamil Nadu

### 👥 Customer Demographics

The dashboard provides insights into customer purchasing behavior based on:

- Gender distribution
- Age groups:
  - Teenager
  - Adult
  - Senior

### 🚚 Order Status Analysis

Orders are categorized into:

- Delivered
- Cancelled
- Returned
- Refunded

### 🛍️ Product Category Analysis

The dashboard supports filtering by categories such as:

- Blouse
- Bottom
- Ethnic Dress
- Kurta
- Saree
- Set
- Top

---

## 📌 Key Insights

- Amazon contributes the highest share of total sales.
- Women account for approximately **64%** of all purchases.
- Adult customers are the largest customer segment.
- Maharashtra generates the highest revenue among all states.
- More than **90%** of orders are successfully delivered.
- Sales peak during the first half of the year.
